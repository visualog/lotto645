# 로또 번호 분석 대시보드 개선이 필요한 점

**확인일:** 2025-10-18

---

## 📌 UI 및 정보 구조

- 데이터 테이블 행이 많지만, **모바일 화면에서 가로 스크롤 및 표시 오류** 발생 가능성 있음
- 각 분석 결과(예: 합계 범위 추천, 합계별 추천 번호)에 대한 **설명 또는 예시 부족**
- 추천 조합(예: 합계 기반 추천, 과거 당첨 합계 추천)의 **추출 근거와 취지 미제공**  
  - 추천 방식이 어떻게 선정된 것인지 이해하기 어려움
- 탭 메뉴간의 네비게이션이 있으나, **각 메뉴 기능에 대한 직관적인 부가 설명 필요**
- **헤더 스타일과 섹션 구분**이 명확하지 않아 정보 분류가 직관적이지 않음

---

## 📊 데이터/시각화 관련

- 모든 추천 번호와 통계가 **텍스트와 간단한 테이블**만 제공되어 **시각적 이해도가 낮음**
- 합계별 추천 등에서 **막대그래프, 분포도, 순위 등 시각화** 추가 필요
- 번호 조합, 추출 과정을 색상 또는 아이콘 등 **시각적 요소**로 표시 필요
- 다른 분석 탭(빈도, 동시 출현 등)에 대해서도 **시각적 비교 그래프 필요**

---

## 📱 모바일/접근성

- 테이블 셀 가독성 저하 (특히 추천 번호 다수 표시 구간)
- **모바일 뷰에서 행/열 정렬 및 스크롤 최적화 부족**
- 모바일 환경에서 탭 클릭 시 **기능 이동/표시 반응성 저하 가능성 있음**

---

## 🛠 기술적/기능적 개선

- 추천 로직과 데이터의 **기준일/업데이트 일자 표기 없음**
- 페이지 내에서 **다크모드, 컬러 테마 변경 등 사용자 설정 기능 미구현**
- 추천 결과 및 분석 내용의 **CSV, XLS, 또는 이미지 내보내기 기능 부재**
- SEO 및 페이지 타이틀, 메타데이터 보강 필요 (검색 및 공유 시 정보 부족)
- **버튼, 링크, 시각적 요소에 대한 키보드 접근성/스크린리더 대응** 미흡

---

## 🔧 추가 제안

- 각 추천 번호 및 통계 결과에 대해 **툴팁이나 상세 설명 영역 추가**
- 추천 번호 개별 복사, 즐겨찾기, 저장 기능 제공 필요
- 데이터와 분석 내역의 **시점 명시 및 실시간/주기적 업데이트 안내** 필요
